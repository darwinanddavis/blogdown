Common blogdown commands

In the YAML header:

draft: true For rendering drafts in local preview mode
publishdate: '2018-03-29' Specify a future date to publish a post. Future posts are only rendered in the local preview mode.
weight: <int> Assign a weight class to order posts that have the same date
slug: A character string as the tail of the URL.

Shortcodes

{{< shortcode-name param >}}
To hide caption: {{< shortcode-name param hidecaption >}}

Paired shortcodes
{{% mdshortcode %}}Stuff to process in the center {{% /mdshortcode %}}
{{< highlight go >}} A bunch of code here {{< /highlight >}}

Shortcode examples

blogdown::shortcode("instagram","Bghj21-Fnu7")
{{% instagram "Bghj21-Fnu7" %}} 
blogdown::shortcode('youtube', 'BWNjjyYFxVx')
{{% youtube "BWNjjyYFxVx" %}} 
blogdown::shortcode('tweet', '852205086956818432')
{{% tweet "852205086956818432" %}}
